Civic Pulse — v0.1 starting atlas

Who governs, and what people already say about it

A first pass at overlaying regime type with the main documented grievance in each country — pulled from existing, publicly available research rather than opinion. This is a seed dataset for ~30 countries to test the idea, not a live feed.

◆ V-Dem Institute (regime classification) ◆ Transparency International CPI (corruption) ◆ Freedom House (civil liberties) ◆ Pew / World Values Survey (public sentiment)

Illustrative starting set, not exhaustive or authoritative — classifications simplify complex, contested realities and can shift quickly. For live, sourced data, the real datasets are public: v-dem.net, transparency.org/cpi, freedomhouse.org, worldvaluessurvey.org. A next step worth considering: swap this seed array for a live CSV pull from those sources.
